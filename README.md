## 基本說明
- 基於ibeacon的室內定位

## 主要功能
- 以應用藍芽4.0模組為基礎，透過多個 ibeacons 信號發射裝置做為定位裝
置，配合3D建模UI呈現於行動裝置上。

## 使用方法
1. 進入主介面後後題是開啟藍芽(若無開啟的話)
2. 按下開始按鈕後則會進行定位
